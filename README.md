# 📦 Solnest Cinematic Director — a free Claude skill from Solnest AI

Paste a listing link and Claude turns it into a polished **20–25 second cinematic walkthrough video**, ready to post — it curates the hero shots, generates the motion, and stitches it together with music.

Built by [Ryan Lefebvre](https://instagram.com/Ryan_Le5) / **Solnest AI**. Free to use. It runs inside the **Claude Desktop app** — you don't write any code.

---

## Install it (2 minutes, no tech skills needed)

### ⚠️ Use the Claude Desktop app — NOT the website
The Claude **website** (in a browser) is just a chat — it can't set anything up on your computer. You need the **Claude Desktop app**, downloaded and signed in: **[claude.ai/download](https://claude.ai/download)**.

### The 5 steps
1. Download this repo (green **Code** button → **Download ZIP**) and unzip it.
2. Open the **Claude Desktop app**, signed in, and start a **new chat**.
3. **Drag the unzipped folder** into the message box.
   *(If it won't let you drag a folder, open it, select everything inside, and drag those files in.)*
4. Paste this and send:
   > **Read the start-here guide in this folder and set it up for me, one step at a time.**
5. Click **Allow** on any permission pop-up — that's just the app letting Claude do the setup.

## What you need (this one has a few requirements)
- **Claude Pro or Max** — it connects 3 tools, and the free plan only allows 1.
- A **Higgsfield** account (a few credits), a free **Firecrawl** account, and a **Descript** account (Hobbyist or higher, so your video isn't watermarked).

*(There's a fuller walkthrough in `1 - START HERE.md` inside the folder if you want to read ahead — but you don't have to.)*

## After it's installed
Open a new chat, paste an Airbnb/VRBO/Zillow link, and say **"make me a Solnest video."** You can't break anything.

---

## What is a "skill"?
A skill is a set of instructions that teaches Claude to do one specific job really well. Once it's installed, Claude just knows how to do it whenever you ask — no setup each time.
